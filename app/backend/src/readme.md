# Backend

## Folder Structure

src/
├── auth0/                  # Auth0-related utilities and middleware
├── controllers/            # Controller definitions for handling route logic
├── middleware/             # Middleware for request processing
├── routes/                 # Route definitions
├── services/               # Services
│   ├── UserSyncService.js  # User sync logic
├── utils/                  # Utility functions (e.g., database connection)
│   ├── db.js               # Database pool connection file
├── app.js                  # Main Express server file
